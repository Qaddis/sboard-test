# Тестовое задание

- ЯП - **TypeScript**;
- Графика - **Canvas API**;
- Сборщик - **Vite**;
- Тесты - **Jest**.

## Запуск

1. Установить все зависимости:

```bash
pnpm i
```

2. Запустить в режиме разработки:

```bash
pnpm dev
```

## Тесты

Тесты можно запустить при помощи:

```bash
pnpm test
```
